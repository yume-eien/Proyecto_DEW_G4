Requisitos del Sistema

Windows

  • Windows 10/11 (64-bit)
  • 2GB de RAM (4GB recomendado)
  • 500MB de espacio en disco
  • Procesador Intel i3/AMD Ryzen 3 o superior
  • Conexión a Internet para la activación

macOS

  • macOS 11.0 Big Sur o superior
  • 4GB de RAM (8GB recomendado)
  • 700MB de espacio en disco
  • Procesador Intel o Apple Silicon
  • Conexión a Internet para la activación

Linux

  • Ubuntu 20.04 LTS o equivalente
  • 4GB de RAM (8GB recomendado)
  • 600MB de espacio en disco
  • Procesador Intel i3/AMD Ryzen 3 o superior
  • Conexión a Internet para la activación
